In order to read the documentation clearly, we need to have the same perception.
Some terms and definitions are provided below for easy reading. If you are unclear, please continue to contact our support team.

| Name               | Description                                                                          |
| ------------------ | ------------------------------------------------------------------------------------ |
| `Operator`         | Partners who want to integrate with **S88Pay** Payment Gateway                       |
| `Payment Provider` | Provider API Payment for Operator                                                    |
| `BO Site`          | Backoffice site, used for Operator to verify transaction                             |
| `Member Site`      | A Website for member to log in and create transaction through S88Pay Payment Gateway |
| `User / Member`    | Customer who create transaction through **S88Pay** Payment Gateway                   |
