### Pre Requirement Provide by Provider

- Server info & API URL
- Credential API : `api_key`, `secret_key`, `merchant_code`
- Credential BO : username, password

### Pre-Requirement Provide by Operator

- [API Callback URL](/callback/page) (mandatory). Used to forward transaction status
- Callback `success url page` (optional).
- Callback `failed url page` (optional)

### Security

- Token and transaction key (Need to create token and key for each transaction). More about generate token [encrypt / decript function](/api-reference/encrypt)
- IP White list for Back Office login
