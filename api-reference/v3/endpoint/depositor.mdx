---
title: "Submit Depositor Account (for KRW only)"
openapi: "POST /api/{merchant_code}/v3/krw-submit-information"
---

<Warning>
  Combine all parameters into one string, separated by `&` character and then
  encrypt using [encrypt_decrypt algorithms](/api-reference/encrypt).
</Warning>

## Parameters

<ResponseField name="transaction_code" type="string" required>
  Generated by the operator. Must be unique for each transaction
</ResponseField>
<ResponseField name="bank_code" type="string" required></ResponseField>
<ResponseField name="account_number" type="string" required>
  Bank Account Number
</ResponseField>


</ResponseExample>
