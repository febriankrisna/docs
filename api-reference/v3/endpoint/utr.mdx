<Warning>
  Combine all parameters into one string, separated by `&` character and then
  encrypt using [encrypt_decrypt algorithms](/api-reference/encrypt).
</Warning>

## Parameters

<ResponseField name="transaction_code" type="string" required>
  Generated by the operator. Must be unique for each transaction
</ResponseField>
<ResponseField name="utr" type="string" required>
  Unique 12-digits number
</ResponseField>

<ResponseExample>

```json 200
{
  "status": "success",
  "message": "Success Submit Utr!"
}
```

</ResponseExample>
